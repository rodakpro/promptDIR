# CreatorLab Free Bundle Starter

This folder contains the **Free Bundle starter for testing**.

Use it as a sandbox bundle for checking dashboard/startpack ideas before they are
promoted into the main CreatorLab Pro site. The current starter app lives in
`dashboard-startpack/`.

## Notes

- This is not the production marketing site.
- Keep experiments isolated here until they are intentionally moved into the
  main app.
- The main site remains in the repository root.
